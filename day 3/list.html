Web Dev Glossary

HTML
HyperText Markup Language — used to create webpages.
CSS
Cascading Style Sheets — controls webpage design and styling.
JavaScript
Programming language used to make webpages interactive.
React
JavaScript library used for building user interfaces.
MERN Stack
MongoDB + Express + React + Node.js.
A full-stack JavaScript framework for building web applications.
API
Application Programming Interface — allows two programs to communicate.
Node.js
Runtime environment used to run JavaScript outside browser.
